Pizzería Bajo Belgrano

Consigna Base

La pizzería bajo belgrano cuenta con muchos combos, y van variando mes a mes. Como programadores de su página web, nos interesa no tener que escribir todas las opciones de combo a mano, si no hacerlo de forma automática por JS.

De los combos, no nos interesa nada más que su número. Para empezar, podemos determinar que son 8 combos. Lo dejamos fijo, pero para cuando aprendamos a buscar información desde datos va a reemplazar este número.

Completar el programa para usando while, el <select id = "combos"> se pueble con 8 <option>. Al hacer click en el botón de id = "button" se debe mostrar el combo elegido en <div id = "elegido">
